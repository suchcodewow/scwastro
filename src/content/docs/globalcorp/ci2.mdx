### Platform Concepts: Templates with Integration

#### A) Create a python template

1. In the left nav, click _Pipelines_.
1. Click the 3 dots on the far right side of _Fastlane-Java_, then click _Clone_.
1. Change the name to `Fastlane-Python`, then click _Clone_.
1. Click on Java Build, then in the middle, click _Overview_.
1. Follow the table to configure:

   ##### Name

   ```
   Python Build
   ```

   ##### Cache Intelligence

   (Enable)

   ![Project Select](/images/continue.png)
   Confirm _Cloud_ infrastructure.
   ![Project Select](/images/continue.png)

1. Click Codebase on the right sidebar
   ![Project Select](/images/codebase.png)
1. Change the repository name to:

   ```
   mainapi-norefs
   ```

   ![Project Select](/images/apply_changes.png)

1. Click on the `Build and Push to GAR` step.
1. Update the Image Name:

```
globalcorp/mainapi
```

![Project Select](/images/apply_changes.png)

1. Click _save_.
