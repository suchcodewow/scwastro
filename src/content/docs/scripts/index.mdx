# Helpers

## Purpose

Helpers are small utilities designed to make installation easier or troubleshoot multi-step processes.

| Helper                    | Purpose                                               |
| ------------------------- | ----------------------------------------------------- |
| Azure API Tester          | Check credentials for regular and gov instances       |
| Azure Extension Installer | Bulk deploy the extension to an entire resource group |
